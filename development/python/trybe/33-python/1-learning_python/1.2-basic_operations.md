# **Operações básicas do Python**

~~~python
2 + 3  # soma: 5
3 - 2  # subtração: 1
2 * 3  # multiplicação: 6
3 ** 2  # exponenciação: 9
25 ** 0.5  # raíz quadrada: 5.0
3 / 2  # divisão float: 1.5
3 // 2  # quociente integer: 1
3 % 2  # resto: 1

my_number = 3 # atribuição de variável
my_number++ # funciona em JavaScript, mas não em Python
my_number += 1 # funciona em Python e em JavaScript

'1' == 1 # comparação: False que seria True em JavaScript
a = [1, 2, 3]
b = [1, 2, 3]
a == b # comparação: True que seria False em JavaScript

and # função do && do JavaScript
or # função do || do JavaScript
not # função do ! do JavaScript

my_number > 2 and my_number < 4 # comparação em Python com estilo JavaScript
2 < my_number < 4 # mesma comparação com estilo Python
~~~
